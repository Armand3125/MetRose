# MetRose

Mini-outil HTML pour explorer un flux GTFS Tisséo et visualiser une ligne en carte ou en schéma.

## Contenu prévu

- Chargement GTFS Tisséo
- Exploration des tables GTFS
- Tracé d’une ligne par `route_short_name`
- Choix du trajet / `shape_id`
- Affichage carte claire
- Affichage schématique adapté mobile
- Indication dynamique des bus actifs selon les horaires GTFS
